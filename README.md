# ScreenOCR for Windows

ScreenOCR allows you to quickly and easily extract text from screen captures via OCR, perfect for extracting text from any on-screen content.

All processing is done **locally on your device** using Windows.Media.Ocr API - completely private and offline!

## Requirements

- Windows 10 version 1809 or later (or Windows 11)
- .NET 8.0 SDK for building the native module
- Raycast for Windows

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Build the Windows OCR module:
```bash
npm run build:windows-ocr
```

3. Run in development mode:
```bash
npm run dev
```

For detailed setup instructions, see the [Windows Setup Guide](./WINDOWS.md)
